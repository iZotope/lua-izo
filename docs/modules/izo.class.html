Module izo.class

iZotope class implmentation.

This module provides a basic class implmentation that allows for single inhertance and constructors.

local class = require "izo.class".class
NewClass = class(nil, {init = function() print 'hello' end})
DerivedClass = class(NewClass, {init = function() print 'top constructor' self.base.init()})
derived_inst = DerivedInst()
