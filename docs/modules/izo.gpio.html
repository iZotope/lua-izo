Module izo.gpio

iZotope gpio class

This module provides a gpio class for manipulating 

local gpio = require "izo.gpio"
gpio_plumbus = gpio.Gpio('plumbus', 'in')
print('The plumbus is '..tostring(gpio_plumbus:get_value())

Functions

Gpio:get_value ()
get the value of the gpio.

get the value of the gpio. Only works if in input state, returns -1 otherwise

Returns:

    number 1 for active, 0 for inactive, -1 if in output state or gpio is invalid
Gpio:init (gpio_name, initial_state)
Gpio constructor.

Parameters:

  • gpio_name string name of the gpio as it appears in /dev/gpio/
  • initial_state string inital state of the gpio pin ('in', 'out', 'high', 'low'). nil to not init
Gpio:set_direction (state)
set the gpio direction.

Parameters:

  • state string the state to set the pin ('in', 'out', 'high', 'low')
Gpio:set_value (value)
set the output value of the gpio.

Set the output value of the gpio. Only works if in output state, ignored otherwise

Parameters:

  • value number 0 for incative, 1 for active
