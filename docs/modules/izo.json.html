Module izo.json

iZotope wrapper for json functionality.

This module wraps cjson and resty.prettycjson as well as providing helper functions

local json = require "izo.json"

Functions

decode (json)
decode a JSON string to a table

Parameters:

  • json string text to be decoded

Returns:

  1. table or nil table of decuded values, or nil on error
  2. error on error, this is the error returned, otherwise nil

Usage:

    mytable, err = decode('{"count" : 5}')
decodefile (path)
decode JSON from a file

Parameters:

  • path string to file to decode

Returns:

  1. table or nil table of decoded valuesm or nil on error
  2. error on error, this is set to the error, otherwise nil

Usage:

    mytable, err = decodefile('/path/to/file.name')
encode (data[, pretty=false])
encode a tabe into a json string

Parameters:

  • data table data to convert into json
  • pretty bool set to true to do pretty string formatting of JSON (default false)

Returns:

  1. string or nil json encoded string, or nil on error
  2. error on error, this is the error returned, otherwise nil

Usage:

    mystring, err = encode({a='hello', 0='test', 1=10})
