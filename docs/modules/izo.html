Module izo

iZotope Lua Standard Libraries.

This module contains a selection of extended stdlib and lua functions as well as various genearl purpose functions.

This module depends on, extends and borrows it's style from the lua stdlib project

izo

In addition to the functions documented on this page, and a version field, references to other submodule functions will be loaded on demand.

Fields:

Metamethods

__index (name)
Lazy loading of izo modules. Don't load everything on initial startup, wait until first attempt to access a submodule, and then load it on demand.

Parameters:

  • name string submodule name

Returns:

    table or nil the submodule that was loaded to satisfy the missing name, otherwise nil if nothing was found

Usage:
